# E-Commerce Data Warehouse for Sales Analytics

## 📌 Project Overview
This project focuses on designing and implementing a Data Warehouse for an E-Commerce system. It uses ETL (Extract, Transform, Load) techniques to process data and generate meaningful business insights.

---

## 🎯 Objectives
- Build a centralized data warehouse
- Perform ETL operations using Python
- Analyze sales data using SQL queries
- Generate reports for decision-making

---

## 🛠️ Technologies Used
- Python
- Pandas
- SQLite
- SQL

---

## 📊 Features
Data extraction from CSV files
Data transformation and cleaning
Data loading into SQLite database
Sales analysis and reporting

---

## 📈 Sample Output
Total Sales
Top Selling Products
Customer Spending Analysis

---

## ✅ Conclusion

This project demonstrates how data warehousing techniques can be used to analyze e-commerce data efficiently and support business decision-making.

---

## 🚀 Future Scope
Integration with real-time data
Cloud-based data warehouse
Dashboard visualization using Power BI / Tableau
